## 这个项目是什么？
使用java语言，山寨java集合框架中的数据结构。
#### 这样做有什么目的?
为了更好的掌握java中的数据结构。


## fast-data有哪些数据结构？

* 自动扩容(缩容)的泛型数组(done)
* 数组队列(done), 数组循环队列(done)
* 链表，跳表，哈希表
* Binary Search Tree
* 集合和映射
* 线段树
* 并查集
* AVL
